# workshop-text-editor
This project explores the creation of a simple text editor.

## Features
- Basic text editing functionalities.
- Save and load text files.

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Use
1. Clone the repository: `git clone https://github.com/jason-mid/workshop-text-editor.git`
2. Open `index.html` in your browser.
